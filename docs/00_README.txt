อัปเดตหลัก v2.4.2

- นำเข้าข้อสอบจริงจาก Blank Result Form โดยคงเลขข้อ โจทย์ รหัสตัวเลือก และข้อความตัวเลือก
- เชื่อม Case Study / Dry Challenge และผลตรวจจาก Kit Instruction เป็นข้อมูลประกอบโจทย์
- เมื่อพบข้อสอบต้นฉบับ ระบบนำเข้าด้วยโค้ดและไม่เรียก AI รอบสร้างคำถามซ้ำ
- คง Workflow ข้อสอบจากภาพผลดิบและข้อสอบที่เพิ่มเองทั้งหมด
- ไม่ลบข้อสอบที่เผยแพร่แล้วหรือข้อสอบที่มีผู้ตอบ
- ใช้ Extraction Cache เดิมของ Raw Result, Antigram, Official Evaluation และ Participant Summary เพื่อลด Token
- คงการติดตั้ง PWA สำหรับ Android, iPhone และ iPad จาก v2.4.1
